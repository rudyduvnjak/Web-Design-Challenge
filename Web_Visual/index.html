Latitude Vs. 4 Weather Elements

This project was used to obtain data from Open Weather API for weather patterns in over 500 cities using Latitude as to obtain list of cities with corresponding weather measurements.

After obtaining data we used Pandas to create data frames, and used Matplotlib to plot four wether elements: Max Temperature Vs Lattitude, Cloudiness Vs. Latitude, Humidity Vs. Latitude, and Wind Speed Vs Latitude.
